# Security Policy

## Supported Versions

Only the latest active release of ApexMemory is supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0.0 | :x:                |

## Reporting a Vulnerability

We take the security and privacy of local workspaces extremely seriously. Since ApexMemory runs entirely on your local machine, safeguarding your files and credentials is our highest priority.

If you discover a security vulnerability or security concern, **please do not open a public issue.** Public disclosure of vulnerabilities puts other users' workspaces at risk.

Instead, please report security vulnerabilities directly to the maintainer via email:
* **Contact:** Security Concerns (please open a private draft or email @AndyTheGuy via GitHub profile)

We will investigate the report immediately, prepare a fix, and coordinate a secure release. Thank you for helping keep our developer community safe!
